Project Euler Problem 53 Solution

Question

There are exactly ten ways of selecting three from five, 12345:


123, 124, 125, 134, 135, 145, 234, 235, 245,  and 345

In combinatorics, we use the notation, 5C3 = 10.

In general,


$$^nC_r = \frac{n!}{r!(n−r)!}, \text{ where } r \leq n, n! = n\times(n−1)\times\dots\times3\times2\times1, \text{ and } 0! = 1.$$

It is not until n = 23, that a value exceeds one-million: 23C10 = 1144066.

How many, not necessarily distinct, values of nCr, for 1 ≤ n ≤ 100, are greater than one-million?

Haskell

choose :: Integer -> Integer -> Integer
choose _ 0 = 1
choose 0 _ = 0
choose n r = choose (n-1) (r-1) * n `div` r

main :: IO ()
main = print $ length $ filter (> 1000000) [n `choose` r | n <- [1..100], r <- [1..n]]

Python

#!/usr/bin/env python
import gmpy
count = 0
for n in range(1, 101):
    for r in range(1, n):
        c = gmpy.comb(n,r)
        if c > 1000000:
            count += 1
print count
